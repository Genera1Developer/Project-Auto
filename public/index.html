**README.md** # Project Goal The goal of this project is to modify and fix the given web proxy to work fully for Vercel and static serverless sites, and fix any bugs along the way. ## File Structure The following is the file structure for this project: - `public/`: - `index.html`: The main HTML file for the web proxy. - `theme/`: - `default.css`: The default CSS styles for the web proxy. - `particles.css`: The CSS styles for the particle effect on the background. - `dropdown.css`: The CSS styles for the dropdown menu. - `visuals/`: - `particles.js`: The JavaScript code for the particle effect on the background. - `src/adblocker/`: - `curseblock.js`: The JavaScript code for the Curse Block ad blocker. - `ublock.js`: The JavaScript code for the uBlock ad blocker. - `vercel.json`: The configuration settings for deploying the project to Vercel. - `package.json`: The configuration settings for the project, including dependencies and scripts. - `README.md`: The documentation for the project. ## Changes and Additions The following changes and additions have been made to the project: - Updated the `script` tag in `index.html` to load the `particles.js` file from the correct path. - Added `curseblock.js` and `ublock.js` files to the `src/adblocker` directory. ## Bug Fixes The following bugs have been fixed in this project: - The particle effect was not working properly. - The ad blockers were not working properly. ## Next Steps The next steps for this project are to: -
