Web Proxy Project

Structured web proxy with strictly defined files and components, baby blue beautiful and rounded theming.

Allowed Files:

- index.html (main landing page)
- settings.html (proxy configuration page)
- dashboard.html (user statistics and monitoring)
- README.md (description of the site)

File Requirements:

1. index.html:

- Login form with username/password fields
- Navigation to settings and dashboard
- Proxy status indicator
- Error message display area
- Connection status display
- External CSS and JS files
- Side bar that goes to /index.html, /dashboard.html, and /settings.html
- Custom CSS styling
- Embedded JavaScript

2. settings.html:

- Proxy configuration form
- Protocol selection (HTTP/HTTPS)
- Port configuration
- Authentication settings
- Bandwidth limits
- Same styling as index.html
- Side bar that goes to /index.html, /dashboard.html, and /settings.html
- Custom CSS styling
- Embedded JavaScript

3. dashboard.html:

- Real-time connection status
- Bandwidth usage graphs
- Active connections list
- Error log display
- User statistics
- Same styling as index.html
- Side bar that goes to /index.html, /dashboard.html, and /settings.html
- Custom CSS styling
- Embedded JavaScript

4. README.md:

- Raw markdown text only

Code Structure Rules:

- No inline scripts or styles
- Semantic HTML elements only
- Strict content security policy
- Error handling for all operations
- Cross-browser compatibility
- Mobile responsiveness
- Clean, consistent indentation
- Proper commenting
- No external dependencies
- No framework usage
