This project is a web proxy that allows users to access websites anonymously.
The project is structured as follows:
- `README.md`: This file contains the project description and instructions.
- `src/`: This directory contains the server-side code.
    - `server.js`: This file contains the main server code.
    - `proxy.js`: This file contains the code for proxying requests.
- `public/`: This directory contains the client-side code.
    - `index.html`: This file contains the main HTML code.
    - `style.css`: This file contains the CSS code.
    - `script.js`: This file contains the JavaScript code.

To use the web proxy, first start the server by running the following command in the project directory:
```
npm start
```
Then, open a web browser and go to the following URL:
```
http://localhost:3000/
```
You can now enter the URL of the website you want to access anonymously in the input field and click the "Go" button. The web proxy will fetch the content of the website and display it in the browser.
