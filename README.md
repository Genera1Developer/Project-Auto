CONTENT: This project implements a fully structured web proxy with the following components:

- `index.html`: User interface for entering URLs and submitting proxy requests.
- `style.css`: Styles the HTML page.
- `proxy.js`: Handles client-side logic, such as form submission and displaying results.
- `server.js`: Node.js server that proxies requests to the specified URL.

How to use:
1. Clone this repository.
2. Install the dependencies using `npm install`.
3. Run the server using `node server.js`.
4. Open `index.html` in a browser and enter the URL you want to proxy.
5. Click the "Proxy" button to retrieve and display the proxied content.
